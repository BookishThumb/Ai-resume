# PROJECT: TALENTAI - AI-Powered Recruitment & Onboarding Platform

Build a complete, production-quality SaaS web application called **TalentAI**.

This is a hackathon project for an AI/ML Full Stack Developer evaluation. The application should solve the complete recruitment lifecycle from candidate application to onboarding.

The application must look like a modern funded startup product and prioritize:

* Exceptional UI/UX
* Smooth animations
* Real AI functionality
* Fast performance
* Mobile responsiveness
* Professional dashboards
* Recruiter workflow automation

The judges will evaluate both functionality and design quality.

---

# JOB MATCHING ENGINE

Compare:

Candidate Resume

VS

Job Description

Use Sentence Transformers:

all-MiniLM-L6-v2

Calculate:

Cosine Similarity

Formula:

Match Score = Cosine Similarity × 100

Display:

* Animated circular score indicator
* Skill overlap chart
* Missing skills analysis
* Match breakdown

---

# CANDIDATE SCORING ENGINE

Calculate Resume Score:

Resume Score =

(0.40 × Skill Match)

*

(0.20 × Project Relevance)

*

(0.20 × Experience Score)

(0.10 × Education Score)
